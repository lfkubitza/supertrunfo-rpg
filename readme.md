# Super Trunfo - ORIGIN RPG

Um jogo estilo Super Trunfo com personagens de RPG feito em linguagem C.

## Classes:
- Guerreiro
- Mago
- Arqueiro
- Assassino
- Paladino
- Curandeiro

## Atributos:
- Força
- Defesa
- Inteligência
- Destreza
- Cura

## Como jogar:
Execute o jogo, veja sua carta e escolha um atributo para competir contra o inimigo.
